== Start Rails

A rails ready setup app for easy go with application development. It includes some trivial steps which we cant avoided for most web apps.

=== App includes:

* Bootstrap setup(<code> gem bootstrap-sass</code>)
* User management:User regsiter/sig in(<code>gem devise</code>)
* Home page
